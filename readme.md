# Crawler

Bài tập thực hành tuần 

![](./image/demo.gif)

## Mô tả

Trương trình giúp cho việc lấy dữ liệu theo danh mục trên trang vnexpress

### Ví dụ

```
npm run vnexpress the-thao
```

### Cài đặt

Required: [Nodejs](https://nodejs.org/en/)

- Clone project
- cd Crawler && npm install
